# CLI TUI 功能规范

## 概述

本文档定义了 MystiSql CLI 的 TUI（文本用户界面）功能规范，包括界面布局、交互方式和基本操作。

## 功能需求

### 1. 命令行接口

- **命令名称**: `mystisql tui`
- **参数**: 可选参数 `--instance` 用于指定初始数据库实例
- **功能**: 启动交互式 TUI 界面

### 2. 界面布局

- **顶部状态栏**: 显示当前连接的数据库实例、数据库名称和连接状态
- **主输入区域**: 用于输入 SQL 命令，支持多行输入
- **结果显示区域**: 显示查询结果，支持表格形式展示
- **底部状态栏**: 显示当前操作状态、快捷键提示

### 3. 基本操作

- **执行 SQL**: 按 Enter 键执行当前输入的 SQL 命令
- **取消输入**: 按 Ctrl+C 取消当前输入
- **退出 TUI**: 按 Ctrl+D 或输入 `exit` 命令退出
- **清屏**: 按 Ctrl+L 清屏

### 4. 命令历史

- **历史记录**: 保存用户执行过的 SQL 命令
- **浏览历史**: 使用上下箭头键浏览历史命令
- **历史持久化**: 将历史记录保存到本地文件

### 5. SQL 语法高亮

- **语法高亮**: 对 SQL 关键字、函数名、字符串等进行语法高亮
- **支持方言**: 支持 MySQL、PostgreSQL、Oracle 等不同数据库的 SQL 方言

### 6. 错误处理

- **错误显示**: 在界面中清晰显示执行错误信息
- **错误定位**: 指出 SQL 语句中的错误位置
- **错误恢复**: 允许用户在错误后继续输入新的 SQL 命令
