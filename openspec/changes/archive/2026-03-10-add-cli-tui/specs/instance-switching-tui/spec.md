# 实例切换 TUI 功能规范

## 概述

本文档定义了 MystiSql TUI 中数据库实例切换功能的规范，包括实例列表展示、切换操作和连接管理。

## 功能需求

### 1. 实例列表

- **实例发现**: 利用现有的实例发现机制获取可用的数据库实例
- **实例分类**: 按数据库类型（MySQL、PostgreSQL、Oracle、Redis）分类显示实例
- **实例信息**: 显示实例名称、类型、状态、地址等信息
- **实例搜索**: 支持搜索和过滤实例

### 2. 实例切换

- **切换方式**: 通过快捷键或菜单选择切换实例
- **连接状态**: 显示连接状态和连接时间
- **自动重连**: 在连接断开时自动尝试重连
- **连接测试**: 切换前测试连接是否可用

### 3. 实例管理

- **实例详情**: 显示实例的详细配置信息
- **实例状态**: 显示实例的健康状态
- **实例监控**: 显示实例的基本监控信息（如连接数、查询数等）
- **实例分组**: 支持对实例进行分组管理

### 4. 连接配置

- **连接参数**: 支持查看和修改连接参数
- **连接超时**: 支持设置连接超时时间
- **连接池**: 支持查看连接池状态
- **SSL 设置**: 支持配置 SSL 连接参数

### 5. 切换历史

- **切换历史**: 记录用户的实例切换历史
- **快速切换**: 支持快速切换到最近使用的实例
- **默认实例**: 支持设置默认实例

### 6. 错误处理

- **连接错误**: 清晰显示连接错误信息
- **实例不可用**: 标记不可用的实例
- **重连策略**: 提供自动重连策略配置
